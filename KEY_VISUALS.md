# Siemens Key Visuals Reference

Detailed reference for Siemens key visual matrix (v1.3.0). See SKILL.md for summary.

## Brand Idea

Tagline: **"WE CREATE TECHNOLOGY TO TRANSFORM THE EVERYDAY FOR EVERYONE"**

### Brand Motifs (Shapes)

Use these motifs in digital key visuals, hero sections, and brand communications. Render in Siemens Petrol or Bold Green on dark backgrounds.

| Motif | Shape | Meaning |
|-------|-------|---------|
| Transformation | Infinity symbol (∞) | Continuous transformation, innovation |
| Moving Forward | Double arrow (» or ⟫) | Progress, forward momentum |
| Siemens Xcelerator | X-shaped icon | Open digital business platform |
| Sustainability | Circle (○) | Circular economy, sustainability commitment |

### Visual Style for Motifs
- Color: Petrol / Bold Green / Light Petrol on Deep Blue backgrounds
- Style: Geometric, clean, minimal strokes
- Animation: Smooth transitions, morphing between states

### Icon Style Guide
- Stroke weight: 1.5px–2px
- Color: `siemens-petrol` or `light-petrol` on `deep-blue` backgrounds
- Size: 24×24px (standard), 48×48px (feature), 64×64px (hero)
- Padding: 4px internal padding within bounding box

## Typography in Key Visuals

- **Font family**: SiemensSans Pro (or system sans-serif fallback)
- **Headlines**: Uppercase, SiemensSans Pro Bold
- **Taglines**: "WE CREATE TECHNOLOGY..." in uppercase, `light-petrol` or `siemens-petrol`
- **Product names**: White, regular weight, on dark backgrounds
- **Sub-labels**: Smaller size, `deep-blue-60` on light or `light-petrol` on dark

## Layout Principles for Key Visuals

1. **Dark backgrounds preferred** for hero sections and brand messaging — use `deep-blue` (`#000028`)
2. **Light backgrounds** for content sections — use `light-sand` (`#f3f3f0`) or `white`
3. **Motifs and icons** always rendered in Petrol/Green/Blue tones, never in secondary colors
4. **Full-bleed images** with dark overlay (`bg-deep-blue/60`) for text readability
5. **Technology icons** in grid layout, consistent sizing, Petrol colored
